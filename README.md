# Linux Homelab

Experimental self-hosted infrastructure focused on:

- Linux administration
- Docker containers
- Monitoring
- Networking
- Remote access
- Self-hosted services
- Automation
- Observability

## Technologies

- Docker
- Portainer
- Linux
- SSH
- Uptime monitoring

## Goals

This project was created for learning and experimentation with self-hosted infrastructure and cloud concepts using low-cost hardware.
